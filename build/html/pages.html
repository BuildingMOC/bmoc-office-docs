1_PM_Procedures.py

Author: Kyle Wengryn Version: 0.0.1

This script adds a page to the BMOC-OFFICE streamlit application. The purpose of this script is to upload adcts and convert them to another excel workbook with 2 sheets: Asset inventory and PMProcedures.

This script uses the BMOC library which can be found at: BMOC - SecureBMOCBMOC_PMLibrary.xlsx
